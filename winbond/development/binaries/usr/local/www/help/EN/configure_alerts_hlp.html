Alerts - Help

On any severity events an alert is generated and sent to the configured user/agent.


Alert can be of two types:-

  • SNMP Trap
  • Email


Actions

Adding/Modifying an alert
To add or modify an alert, select the slot and click 'Modify' button. This will redirect alert modification page where the alert can be configured.

Send Test Alert
To check whether a configured alert is working, select a slot which is configured and click 'Send Test Alert' button. This will generate a test alert to the configured user/agent.

Deleting an alert
To delete an alert, select the slot and click 'Delete' button. A notification will be given on the status of the delete action.
