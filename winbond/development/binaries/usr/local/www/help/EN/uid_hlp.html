UID - Help

Redirection Console handles mouse emulation from local window to remote screen in two methods.
  • UID On
  • UID Off

UID On

The absolute position of the local mouse is sent to the server. To select this mode click on "Set mode to On" radio option.

UID Off

Relative mode sends the calculated relative mouse position displacement to the server. To select this mode click on "Set mode to Off" radio option.
Actions

Save
