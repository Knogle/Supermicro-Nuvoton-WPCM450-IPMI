Adding User - Help

Use this page to add a new user.

User Name

The name of the new user.
  • - User Name is a string of 4 to 15 alpha-numeric characters.
  • - It must starts with an alphabet.

Password, Confirm Password

Enter and confirm the new password here.
  • - Password must be atleast 8 characters long.

Network Privileges

The level of network privilege to assign to this user.


Actions

Add

Click on Add button to save the new user and return to the user list.

Cancel

Click on cancel to return to the list.
