SMTP - Help

Enter the IP Address for the SMTP Mailserver and press Save.

Mail Server IP

The IP address of the SMTP Mailserver.

  • - IP Address made of 4 numbers seperated by dots as in "xxx.xxx.xxx.xxx".
  • - Each Number ranges from 0 to 255.
  • - First Number must not be 0.

Actions

Save

Click on 'Save' button to save the new SMTP sever IP . To delete the SMTP Server configuration, empty Mail Server IP text box and click on 'Save' button
