Alert Modification - Help

Alert Type Field

Alert can be either sent as SNMP Trap or Email alert.

For Email alert, the fields like Email address, subject and body of the message has to be filled.

For SNMP Trap, the destination IP address has to be filled.

Event Severity

Select the severity level of this alert.

Destination IP

The IP address of the system that will receive the alert.
  • - IP Address made of 4 numbers seperated by dots as in "xxx.xxx.xxx.xxx".
  • - Each Number ranges from 0 to 255.
  • - First Number must not be 0.

Email Address, Subject & Message

These fields must be configured if email alert is chosen as alert type. An email will be sent to this email address in case of any severity events with a subject specified in subject field and will contain the message field's content as the email body.
Actions

Save

Click on 'Save' button to save the alert configuration. Status of the action performed will be notified.


Cancel

Click on 'Cancel' button to cancel any changes made and go back to alert list page.
