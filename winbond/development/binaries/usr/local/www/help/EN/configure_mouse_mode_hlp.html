Mouse Mode - Help

Redirection Console handles mouse emulation from local window to remote screen in two methods.
  • Absolute
  • Relative

Absolute Mode

The absolute position of the local mouse is sent to the server. To select this mode click on "Set mode to Absolute" radio option.

Relative Mode

Relative mode sends the calculated relative mouse position displacement to the server. To select this mode click on "Set mode to Relative" radio option.
Actions
