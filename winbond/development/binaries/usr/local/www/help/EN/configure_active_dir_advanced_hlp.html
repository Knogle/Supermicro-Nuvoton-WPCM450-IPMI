Active Directory Advanced Settings - Help

Use this page to configure Active Directory Advanced Settings

Enable Active Directory Authentication

To enable/disable Active Directory, check or uncheck the "Enable LDAP Authentication" checkbox respectively.

Check the box below to enable Active Directory authentication and enter the required information to access the Active Directory server.

User Domain Name

Specify the Domain Name for the user

Time Out

Specifies the time (in seconds) to wait for Active Directory queries to complete.

  • - Default Time out value is 120 seconds.
  • - Range between 15 and 300 allowed.

Domain Controller Server Address1, Domain Controller Server Address2 & Domain Controller Server Address3

The IP address of Active Directory server. Atleast one Domain Controller Server Address must be configured.

  • - IP Address made of 4 numbers seperated by dots as in "xxx.xxx.xxx.xxx".
  • - Each Number ranges from 0 to 255.
  • - First Number must not be 0.

Active Directory Certificate

If no certificate is uploaded, then it will use the default Active Directory Certificate.

Upload New Active Directory Certificate

Browse the Active Directory Certificate file using Browse button.


Actions

Save

Save the current settings.


Cancel

Click Cancel button to cancel modifications and return to the Active Directory Main page.
