Network - Help

Use this page to configure the network settings.

MAC Address

The MAC address of the device (read only).

IP Address

Select the type of IP assigment with the radio buttons.  If configuring a static IP, enter the requested address, subnet mask, and gateway in the given fields.

  • - IP Address made of 4 numbers seperated by dots as in "xxx.xxx.xxx.xxx".
  • - Each Number ranges from 0 to 255.
  • - First Number must not be 0.

Enable VLAN

Check the box below to enable Virtual LAN(VLAN) and enter the VLAN Tag.

VLAN Tag

VLAN Tag(VLANID) of the VLAN.

  • - VLAN Tag should contains only numericals.
  • - It allows range between 2 and 4095. (Value '1' is default VLAN tag, and cannot be allowed.)

Actions

Save

Click on 'Save' button to save any changes made.
