Adding Role Group - Help

Use this page to add a new role group.

Role Group Name

The name that identifies the role group in the Active Directory
  • - Role Group Name is a string of 255 alpha-numeric characters.
  • - Special symbols hyphen and underscore are allowed.

Role Group Domain Name

The domain where the role group is located.
  • - Domain Name is a string of 255 alpha-numeric characters.
  • - Special symbols hyphen, underscore and dot are allowed.

Role Group Privilege

The level of privilege to assign to this role group.


Actions

Add

Click on Add button to save the new role group and return to the Role Group list.

Cancel

Click on cancel to return to the Role Group list.
